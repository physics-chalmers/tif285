Course literature

  • Course textbook: "Bayesian Logical Data Analysis for the Physical Sciences", by Phil Gregory, 2005. The paperback version (from 2010) should be for sale at Store, Chalmers. An electronic version of the book is available through books24x7 via Chalmers library (login required):
    • Phil Gregory, "Bayesian Logical Data Analysis for the Physical Sciences", Cambridge University Press (2005).
  • Lecture notes. See below.
  • Additional reading:
    1. David J.C. MacKay, "Information Theory, Inference, and Learning Algorithms", Cambridge University Press (2005).
    2. D.S. Sivia, "Data Analysis : A Bayesian Tutorial", Oxford University Press (2006).
    3. Trevor Hastie, Robert Tibshirani, Jerome H. Friedman, "The Elements of Statistical Learning", Springer (2009).
    4. Andrew Gelman et al., "Bayesian Data Analysis", CRC Press, 3rd edition (2014). e-book not available.
    5. Aurelien Geron, "Hands‑On Machine Learning with Scikit‑Learn and TensorFlow", O'Reilly (2017).
    Electronic versions of some of these books are available via the author or via Chalmers library (login required). See links above.
